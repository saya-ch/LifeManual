## 这是什么？

LifeManual 是一个温暖而深刻的「人生说明书生成器」。

它不是星座运势，不是MBTI标签，不是心理测试。

它通过 **5-7个精心设计的问题** 了解你的内在世界，然后为你生成一本**专属于你的、精美的、可以直接截图分享的人生说明书**。

> "如果明天全世界停电一天，你会做什么？"

从一个问题开始，到一本属于你的说明书。

## 长什么样？

生成的「人生说明书」是一本精美的HTML页面，包含：

| 章节 | 内容 |
|------|------|
| **封面** | 你的名字，生成日期，"全球限量一册" |
| **关于这个人** | 气质类型、核心驱动力、隐藏的矛盾 |
| **内在光谱** | 理性/感性、独处/社交等5个维度的可视化分析 |
| **能量说明书** ⭐ | 能量来源、消耗、一级省电模式 |
| **精神避难所** ⭐ | TA的精神避难所和避难仪式 |
| **隐藏技能** | 你可能自己都没意识到的3-4个能力 |
| **清醒时刻** ⭐ | 人生节点，给十年前的自己 |
| **使用建议** | 如何与你相处的4条建议 |
| **注意事项** | 关于你的2-3个重要提醒 |
| **保修条款** | "保修期：终身" |
| **抽象版自我介绍** ⭐ | 小红书"抽象力"风格的自我描述 |

设计风格：温暖的大地色系，大量留白，像一本精心排版的独立杂志。

## 使用方法

### 方式一：在TRAE SOLO中导入

1. 下载 [`SKILL.md`](SKILL.md)
2. 在TRAE中打开 **设置 → Rule & Skills → 导入**
3. 对它说："帮我生成一本人生说明书"

### 方式二：直接对话

导入后，直接和它聊天。它会问你5-7个问题，然后生成你的专属说明书。

## 设计理念

### 为什么是"说明书"？

每个人都在"使用"自己，但很少有人认真读过自己的说明书。

我们给手机读说明书，给新买的家具读说明书，却从来没给自己的生命读过说明书。

### 为什么是HTML？

因为说明书应该是**美的**。

它不应该是一段文字，而应该是一个**可以保存、可以分享、可以反复翻阅的物件**。

HTML让说明书拥有了排版、配色、留白——拥有了"物"的质感。

### 气质类型命名

不使用MBTI或星座术语，用更有画面感的诗意描述：

- "深夜电台主持人型" — 声音不大，但让人想一直听下去
- "雨天咖啡馆型" — 安静但有温度，适合长时间待着
- "旧书店型" — 看起来不起眼，但越翻越有东西
- "公路电影型" — 看似随性，其实有自己的路线
- "手写信型" — 在快节奏的时代里，选择慢慢来
- "凌晨三点型" — 别人都睡了的时候，才刚开始
- "一级省电模式型" — 平时不显山露水，关键时刻能量惊人
- "精神避难所型" — 总有一个地方，在那里可以完全不用装
