## 这是什么？

LifeManual 是一个温暖而深刻的「人生说明书生成器」。

它不是星座运势，不是MBTI标签，不是心理测试。

它通过 **5-7个精心设计的问题** 了解你的内在世界，然后为你生成一本**专属于你的、精美的、可以直接截图分享的人生说明书**。

> "如果明天全世界停电一天，你会做什么？"

从一个问题开始，到一本属于你的说明书。

## 长什么样？

生成的「人生说明书」是一本精美的HTML页面，包含10个章节 + 活人感自我介绍：

| 章节 | 内容 |
|------|------|
| **封面** | 你的名字，生成日期，"全球限量一册" |
| **关于这个人** | 气质类型、核心驱动力、隐藏矛盾、气血强度 |
| **内在光谱** | 理性/感性、独处/社交等5个维度的可视化分析 |
| **能量说明书** | 能量来源、消耗、一级省电模式 |
| **精神角落** | TA的精神角落、进入仪式、通勤式疗愈 |
| **隐藏技能** | 人类丰容术、有效努力术等3-4个能力 |
| **清醒时刻** | 人生节点，给十年前的自己 |
| **人生丰容清单** ⭐ | 本周推荐、通勤式疗愈、精神角落升级、敬自己一杯 |
| **使用建议** | 如何与你相处的4条建议 |
| **注意事项** | 关于你的2-3个重要提醒 |
| **保修条款** | "保修期：终身" |
| **活人感自我介绍** | 不是标签，不是人设，有毛边但鲜活的自己 |

## 为什么做这个？

2026年5月，年轻人正在经历一场从"向外求取"到"向内生长"的深刻转变。

- **62.8%** 青年处于中高压力状态（《2026青年情绪白皮书》）
- **77.3%** 期待AI帮助自己"更好地认识自己"（Soul×复旦报告）
- **5.3亿+** 浏览"人类丰容"话题，年轻人渴望给日常制造小变化
- AI陪伴用户规模突破 **5亿**，日均交互超2小时

LifeManual 不给你贴标签，它帮你**描述自己**——用有温度的语言、有毛边但鲜活的真实感。

## 使用方法

### 方式一：在TRAE SOLO中导入

1. 下载 [`SKILL.md`](SKILL.md)
2. 在TRAE中打开 **设置 → Rule & Skills → 导入**
3. 对它说："帮我生成一本人生说明书"

### 方式二：直接对话

导入后，直接和它聊天。它会问你5-7个问题，然后生成你的专属说明书。

## 设计理念

### 为什么是"说明书"？

每个人都在"使用"自己，但很少有人认真读过自己的说明书。

### 为什么是HTML？

因为说明书应该是**美的**——可以保存、可以分享、可以反复翻阅的物件。

### 为什么是"活人感"？

2026年的年轻人不需要另一个完美的标签，他们需要被**真实地看见**——包括毛边、包括矛盾、包括"如何呢又能怎"的无奈。
