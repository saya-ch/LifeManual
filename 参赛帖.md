# SOLO技能创作赛参赛帖

## 📖 LifeManual — 人生说明书生成器

> "每个人都是一本还没被好好读过的书。这本说明书，只是一个开始。"

---

## 一、项目简介

**LifeManual** 是一个温暖而深刻的「人生说明书生成器」。

它不是星座运势，不是MBTI标签，不是心理测试。

它通过 **5-7个精心设计的问题**，了解你的内在世界，然后为你生成一本**专属于你的、精美的、可以直接截图分享的人生说明书**。

### 核心价值

- **情感价值**：帮用户被"真正看见"，而不是被分类
- **社交货币**：生成的HTML页面精美可分享，适合小红书/朋友圈传播
- **自我认知**：通过对话式问答，引导用户进行深度自我探索

## 二、创意来源

### 2.1 市场洞察

基于深度调研发现：

| 数据来源 | 关键发现 |
|---------|---------|
| 小红书2026十大热词 | 抽象力(450亿+)、痛文化(143亿+)、主体性、活人感、边界感、柔软力 |
| 青年心理报告 | 64.2%青年存在精神内耗，63%年轻人未参与线下聚会 |
| Z世代趋势 | "一级省电模式"成为流行表达，年轻人渴望被理解但不愿被标签化 |

### 2.2 差异化定位

现有SOLO Skills多为**工具型**（小红书检测、基金分析等），缺乏**情感价值型**。

LifeManual填补了这个空白：
- 不给用户贴标签，而是帮用户**描述自己**
- 不输出冷冰冰的分析报告，而是生成**有温度的视觉作品**
- 不问标准化问题，而是进行**像老朋友聊天一样的对话**

### 2.3 设计理念

灵感来自深夜和老朋友喝酒时，突然说了一句让你沉默很久的话。

**核心原则**：
1. 用场景代替形容词
2. 发现矛盾之美
3. 避免鸡汤，要说具体的事
4. 适度幽默，但从不轻浮

## 三、功能特点

### 3.1 对话式问答设计

问题分为6个层次，从轻松到深入：

| 层次 | 示例问题 | 目的 |
|------|---------|------|
| 破冰层 | "如果明天全世界停电一天，你会做什么？" | 轻松开场，降低心理防备 |
| 日常层 | "周末对你来说意味着什么？" | 了解生活方式 |
| 能量层 | "什么事情做完之后，你会觉得'活过来了'？" | 探索能量来源 |
| 精神避难所层 | "你有没有一个地方，在那里可以完全不用装？" | 发现内心避风港 |
| 价值观层 | "你觉得'成功'对你来说意味着什么？" | 触及核心信念 |
| 清醒时刻层 | "如果此刻可以给十年前的自己写一句话，你会写什么？" | 引导深度反思 |

### 3.2 说明书结构（9个章节）

生成的「人生说明书」包含：

1. **封面** — 用户昵称，"全球限量一册"
2. **关于这个人** — 气质类型、核心驱动力、隐藏的矛盾
3. **内在光谱** — 5个维度的可视化分析
4. **能量说明书** ⭐ — 能量来源、消耗、省电模式
5. **精神避难所** ⭐ — TA躲在哪，避难仪式是什么
6. **隐藏技能** — 3-4个用户可能没意识到的能力
7. **清醒时刻** ⭐ — 人生节点，给十年前的自己
8. **使用建议** — 如何与这个人相处
9. **注意事项** — 关于这个人的重要提醒
10. **保修条款** — "保修期：终身"
11. **抽象版自我介绍** ⭐ — 小红书"抽象力"风格

## 四、使用方法

### 方式一：在TRAE SOLO中导入

1. 下载 [`SKILL.md`](https://github.com/saya-ch/LifeManual/blob/main/SKILL.md)
2. 在TRAE中打开 **设置 → Rule & Skills → 导入**
3. 对它说："帮我生成一本人生说明书"

### 方式二：直接对话

导入后，直接和它聊天。它会问你5-7个问题，然后生成你的专属说明书。

## 五、技术亮点

### 5.1 零依赖输出

- 生成的HTML包含**完整的内联CSS**
- 无需任何外部资源，可直接在浏览器打开
- 响应式设计，适配手机和桌面

### 5.2 可分享性

- 截图即可分享到小红书、朋友圈等
- 设计风格参考独立杂志排版美学
- "全球限量一册"的概念增强独特感

### 5.3 气质类型命名

不使用MBTI或星座术语，而是用更有画面感的诗意描述：

- "深夜电台主持人型"
- "雨天咖啡馆型"
- "旧书店型"
- "公路电影型"
- "手写信型"
- "凌晨三点型"
- "一级省电模式型"
- "精神避难所型"

## 六、示例展示

### 对话过程示例

```
🤖 如果明天全世界停电一天，你会做什么？

👤 我可能会出去走走，看看没有灯光的城市是什么样的。
   然后回家，点一根蜡烛，给几个重要的人写一封信。

🤖 你说"给几个重要的人写信"——你心里已经有具体的人了对吗？
   那封信，你最想对谁说些什么？

👤 对我妈吧。想告诉她我其实过得挺好的，
   让她别担心。虽然有时候确实不太好……

🤖 "虽然有时候确实不太好"——这句话你平时不会说出来吧？

👤 嗯……不太会说。觉得说了也没用，还让人担心。

🤖 我明白了。最后一个问题：
   如果此刻可以给十年前的自己写一句话，你会写什么？

👤 "你以后会没事的。真的。"
```
