每个人都不是单维度的。以下是这个人的内在光谱分析。

理性 ← → 感性

你比看起来要感性得多。你只是学会了用理性来包装感性——"没事"其实是有事，"随便"其实不随便。

独处 ← → 社交

你需要在独处中恢复能量。社交对你来说是"支出"，不是"收入"。

计划 ← → 随性

你心里有计划，但不会说出来。你更喜欢"做到了再说"，而不是"说了再做"。

理想主义 ← → 现实主义

你心里住着一个理想主义者，但表现出来的都是现实主义者。这是你的保护色。

敏感 ← → 钝感

你非常敏感。你能感知到别人没说出口的情绪，但你学会了假装没感觉到——因为"感觉到了又怎样呢"。
